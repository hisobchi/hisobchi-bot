# Hisobchi Bot 🤖

Telegram бот для Hisobchi.uz — управление финансами прямо из Telegram.

## Возможности

- 📊 Сводка за сегодня (приход, расход, остаток)
- ➕ Добавить приход через меню
- ➖ Добавить расход через меню
- ⚡ Быстрый ввод текстом: `+ 500000 Выручка от продаж`
- 📥 Импорт Excel файла прямо в чат
- 📋 Последние 10 операций
- 🏢 Переключение между компаниями

## Деплой на Render.com (бесплатно)

### Шаг 1 — Создать бота
1. Напишите @BotFather в Telegram
2. `/newbot` → введите имя и username
3. Скопируйте токен

### Шаг 2 — Загрузить на GitHub
1. Создайте новый репозиторий `hisobchi-bot`
2. Загрузите все файлы из этой папки

### Шаг 3 — Задеплоить на Render
1. Зайдите на render.com → Sign Up (через GitHub)
2. New → Web Service
3. Выберите репозиторий `hisobchi-bot`
4. Настройки:
   - Name: `hisobchi-bot`
   - Runtime: `Python 3`
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python bot.py`
5. Environment Variables → Add:
   - `BOT_TOKEN` = ваш токен от BotFather
   - `SUPABASE_URL` = https://zykqqqtjedsvegtvrapz.supabase.co
   - `SUPABASE_KEY` = ваш anon key
6. Нажмите **Create Web Service**

### Готово! 🎉

## Быстрый ввод операций

```
+ 500000 Выручка от продаж     → добавит приход 500,000 сум
- 150000 Аренда офиса          → добавит расход 150,000 сум
- 50000 Зарплата               → добавит расход, статья "Зарплата"
```
